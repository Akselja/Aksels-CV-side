Ferdigheter

Mye jeg har lært fikk jeg fra Angela Yu sitt Udemy kurs, der har jeg lært mange ting blandt annet: En god del HTML, CSS og JS, Git og Github, og hvordan man kan bruke dem sammen, kommandolinje, npm pakker og servere ved bruk av express, hvordan API-er virker, og bittelitt grunnleggende om databaser.
